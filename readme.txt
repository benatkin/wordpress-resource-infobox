Provides a shortcode for displaying an infobox about a resource.

== Description ==

This plugin provides a shortcode for displaying an information box about
a resource. It takes a resource URL and a description URL containing
display rules, both in JSON.  It can also pull down a description
automatically if a header is set.

== Installation ==

Use the standard WP plugin installation procedure.
